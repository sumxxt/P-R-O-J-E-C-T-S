Title : EDA on Bank Loan Decision Making

Platform : Python(pandas,numpy), Data Visualization(matplotlib, Seaborn)

Duration : 1 Month

Project Description : 

EDA on Bank Loan Decision Making project is to develop an exploratory data analysis (EDA) solution that helps banks and financial institutions in making informed loan decisions. The EDA solution should help in identifying the key factors that influence loan approval, understanding the relationships between these factors with defaulters and repayers.

Dataset : https://www.kaggle.com/datasets/gauravduttakiit/loan-defaulter
